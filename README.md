# 💻 Trabalho Acadêmico em C++ – Programação Estruturada

Este projeto foi desenvolvido como atividade avaliativa da disciplina de Programação, contendo documentação teórica (arquivo .docx) e implementação prática em C++.

O trabalho demonstra a aplicação completa dos fundamentos de lógica de programação, estruturação algorítmica e boas práticas de organização e documentação de código.

---

## 🎯 Objetivo do Projeto

Aplicar na prática os conceitos fundamentais da programação estruturada utilizando C++, incluindo:

- Construção de algoritmos
- Organização lógica do fluxo do programa
- Estruturas condicionais (if / else)
- Estruturas de repetição (for / while)
- Manipulação de variáveis
- Entrada e saída de dados
- Validação de informações
- Comentários explicativos no código
- Documentação formal do projeto

---

## 🛠 Tecnologias Utilizadas

- Linguagem C++
- Programação estruturada
- Compiladores compatíveis (g++, Dev-C++, Code::Blocks, etc.)
- Documentação em formato .docx

---

## 📂 Estrutura do Repositório

- Arquivo principal `.cpp`
- Código totalmente comentado explicando cada etapa do processo
- Documento `.docx` contendo explicação teórica e descrição do funcionamento do algoritmo

## 🧠 Conceitos Aplicados no Código

- Declaração e manipulação de variáveis
- Operadores aritméticos e lógicos
- Estruturas de decisão
- Estruturas de repetição

## ▶ Como Executar

### 🔹 Compilando via Terminal (g++)

1. Acesse a pasta do projeto:
   ```
   cd caminho/do/projeto
   ```
2. Compile o programa:
   ```
   g++ nome_do_arquivo.cpp -o programa
   ```
3. Execute:
   ```
   ./programa
   ```

### 🔹 Utilizando IDE

1. Abra o arquivo `.cpp` no Dev-C++, Code::Blocks ou similar.
2. Compile.
3. Execute o programa.
